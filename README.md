# Custom AI Chatbot

A simple command-line AI chatbot using OpenAI's GPT model.

## Setup

1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2. Add your OpenAI API key to `chatbot.py`.

3. Run the chatbot:
    ```bash
    python chatbot.py
    ```

Type 'exit' to quit the chatbot.
